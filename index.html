Resistencia Eletrica de um Condutor

É a capacidade que um corpo qualquer tem de se opor à passagem de corrente elétrica por si, quando existe uma diferença de potencial aplicada. A unidade de medida utilizada para a resistência elétrica é o Ohm, simbolicamente representado pela letra grega (ômega)

Calcule a resistência elétrica de um condutor de cobre com 100 m de comprimento e 1,5 mm2 de secção transversal, usando a formula ea tabela como referencia - Formula: R = ρ * L / S
R =?
ρ = 0,017
L= 100 m
S = 1,5 mm2

Grandezas elétricas

Grandezas elétricas: Unidades de medida do Sistema Internacional também define as unidades de medida para essas grandezas elétricas, que são:
Tensão = Volts (V); Corrente = Ampere (A); Potência = Watt (W); Resistência = Ohm (Ω)

Faça as conversões dos valores para a unidade mais adequada.

  • c) 5780 mA ==> A
